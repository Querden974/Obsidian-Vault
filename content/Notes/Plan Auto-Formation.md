# 🧑‍💻 Programme d'apprentissage – Maintenance Informatique N1/N2

Objectif : Être opérationnel dès les premières semaines de ton alternance.

---

<!-- Column 1 -->
## 🔧 Semaine 1 : Fondations et Outils de base (Poste utilisateur & réseau)

### 🎯 Objectifs :

- Comprendre l’environnement Windows (poste client)
- Dépanner les problèmes courants (niveau 1)
- Connaître les outils réseau et leur usage

### 📚 À apprendre / pratiquer :

- [**Active Directory**](/ef6337bf453e4d36b567d19273dc40ae)
    - Créer un utilisateur
    - Réinitialiser un mot de passe
    - Ajouter un utilisateur à un groupe
- [**Commandes Windows utiles**](/398962b9c7d549718be31e0472f009fa)[ : ](/398962b9c7d549718be31e0472f009fa)
    - `ipconfig`, `ping`, `nslookup`, `tracert`, `netstat`, `tasklist`, `net use`, `whoami`
- [**Utilitaires**](/84d2a7aef33149c996c167c1246b65a4) :
    - `services.msc`, `msconfig`, `taskmgr`, `regedit`, `eventvwr`
- **Gestion des périphériques :**
    - Pilotes, imprimantes, audio, réseau
- **Réflexes de support de niveau 1 :**
    - Identifier le problème (périmètre, reproductibilité)
    - Vérifier les connexions, câbles, périphériques
    - Redémarrer proprement

---

## 📦 Semaine 3 : Déploiement et gestion de parc

### 🎯 Objectifs :

- Comprendre les outils de déploiement (OS, logiciels)
- Dépanner une installation ou configuration automatique

### 📚 À apprendre / pratiquer :

- **MDT (Microsoft Deployment Toolkit)** :
    - Créer une image système
    - Task Sequence de base
    - Logs : `C:\\MININT\\SMSOSD\\OSDLOGS`
- **SCCM – approfondir** :
    - Collections, packages
    - Déploiement ciblé
    - Mises à jour Windows
- **Intune (si environnement cloud)** :
    - Inscription d’un poste
    - Déploiement de profils/configurations
    - Suivi de conformité

<!-- Column 2 -->
## 🖥️ Semaine 2 : Outils de gestion et ticketing

### 🎯 Objectifs :

- Être à l’aise avec le ticketing et les processus ITIL
- Utiliser les outils de gestion de parc
- Comprendre les workflows d’assistance

### 📚 À apprendre / pratiquer :

- [**ServiceNow**](https://www.youtube.com/watch?v=pqKc74DumAI) :
    - Créer / suivre / résoudre un ticket
    - Catégoriser (incident, demande)
    - Ajouter des notes, assigner, clôturer
- [**Jira**](https://www.youtube.com/watch?v=8jWKwiIcWPI) :
    - Créer un projet test
    - Utiliser les statuts, workflows simples
- [**SCCM**](https://www.youtube.com/playlist?list=PLpAbApdrZShNNSOkD3uyA5PLMoXmWPhM7)** – premières bases** :
    - Naviguer dans la console
    - Identifier une machine
    - Déploiement manuel
    - Logs : `C:\\Windows\\CCM\\Logs`
- **VPN – GlobalProtect** :
    - Connexion / déconnexion
    - Résolution d’erreurs (auth, DNS, pare-feu)

## 🧠 Semaine 4 : Scénarios complets + comportement pro

### 🎯 Objectifs :

- Gérer une intervention complète de A à Z
- Adopter un comportement professionnel (communication, documentation)

### 📚 Cas pratiques :

- Réinitialisation de mot de passe + ticket
- Déploiement logiciel via SCCM + suivi
- Connexion VPN distante + test accès réseau
- Dépannage impression + mise à jour du ticket
- Ajout d’un utilisateur AD + groupe + vérification

### 🧾 Soft Skills :

- Poser les bonnes questions à l’utilisateur
- Utiliser un ton calme et professionnel
- Documenter chaque intervention
- Remonter tout comportement anormal ou hors périmètre
