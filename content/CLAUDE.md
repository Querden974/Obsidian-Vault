# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

This is an **Obsidian vault** containing notes for a TSSR (Technicien Supérieur Systèmes et Réseaux) apprenticeship. The content is written in French and covers IT infrastructure, networking, scripting, and system administration.

## Vault Structure

- `Notes/_Notes.md` — master index/dashboard linking all topics
- `Notes/Plan Auto-Formation.md` — 4-week onboarding self-study plan
- `Suivi Alternance.md` — apprenticeship application tracking table
- `Bases des Réseaux Informatiques.md` — networking fundamentals

## Obsidian conventions used

- **Internal links**: `[[Note Name]]` syntax for cross-references
- **Callouts**: `> [!note]+` blocks for collapsible sections
- **Multi-column layout**: `--- start-multi-column` / `--- end-multi-column` blocks (Obsidian plugin)
- **Frontmatter**: YAML `---` blocks at the top of files (mostly empty currently)
- **Embeds**: `![[filename]]` to embed images or other notes

## Content language

All notes are written in **French**. When editing or adding content, maintain French unless the technical term is inherently English (CLI commands, tool names, etc.).
