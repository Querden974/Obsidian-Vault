- **TCP (Transmission Control Protocol)** : Protocole de communication qui garantit la transmission fiable des données entre deux machines (contrôle d’erreurs, ordre des paquets). Utilisé par exemple pour le web (HTTP/HTTPS), les mails (SMTP, IMAP), etc.
- **UDP (User Datagram Protocol)** : Protocole de communication sans garantie de livraison ni d’ordre des paquets, mais plus rapide que TCP. Utilisé pour la voix (VoIP), la vidéo en streaming, ou les jeux en ligne.
- **IP (Internet Protocol)** : Protocole qui identifie les machines sur un réseau grâce à une adresse unique (IPv4 ou IPv6) et assure le routage des paquets.
- **DNS (Domain Name System)** : Service qui traduit les noms de domaine (ex. *openai.com*) en adresses IP compréhensibles par les machines.
- **DHCP (Dynamic Host Configuration Protocol)** : Service qui attribue automatiquement une adresse IP et d’autres paramètres réseau (passerelle, DNS, masque) aux machines d’un réseau.
- **NAT (Network Address Translation)** : Technique qui permet de faire correspondre plusieurs adresses IP privées d’un réseau local à une seule adresse IP publique (utilisé dans les box Internet).
- **VPN (Virtual Private Network)** : Connexion chiffrée qui crée un "tunnel" sécurisé entre un utilisateur et un réseau distant (ex. accéder au réseau de son entreprise à distance).
- **LAN (Local Area Network)** : Réseau local couvrant une zone restreinte (ex. maison, entreprise).
- **WAN (Wide Area Network)** : Réseau étendu couvrant de grandes distances (ex. Internet).
- **Firewall (pare-feu)** : Dispositif (logiciel ou matériel) qui contrôle le trafic réseau entrant/sortant selon des règles de sécurité.
- **Proxy** : Serveur intermédiaire qui relaie les requêtes entre un client et Internet (filtrage, cache, anonymat).
- **VLAN (Virtual LAN)** : Réseau local virtuel permettant de segmenter logiquement un réseau physique pour séparer les flux.
- **SSH (Secure Shell)** : Protocole de communication sécurisé permettant d’administrer à distance des serveurs (connexion en ligne de commande).
- **FTP (File Transfer Protocol)** : Protocole utilisé pour transférer des fichiers entre un client et un serveur.
- **HTTP/HTTPS (HyperText Transfer Protocol / Secure)** : Protocole de communication utilisé pour accéder aux pages web ; HTTPS ajoute le chiffrement (SSL/TLS).
- **SMTP, IMAP, POP3** : Protocoles de messagerie.
    - **SMTP** : envoi de mails
    - **IMAP / POP3** : réception de mails
