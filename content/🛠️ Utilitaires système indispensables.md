| Utilitaire                        | Commande d’accès | Description courte                                                 |
| --------------------------------- | ---------------- | ------------------------------------------------------------------ |
| **Services**                      | `services.msc`   | Affiche tous les services Windows (démarrage, arrêt, statut)       |
| **Éditeur de registre**           | `regedit`        | Accès à la base de registre (avancé – à manipuler avec précaution) |
| **Gestionnaire des tâches**       | `taskmgr`        | Permet de voir les processus, la mémoire, le réseau, etc.          |
| **Configuration système**         | `msconfig`       | Outil pour gérer le démarrage, services, boot Windows              |
| **Visionneuse d’événements**      | `eventvwr`       | Pour consulter les logs système (erreurs, avertissements, infos)   |
| **Gestionnaire de périphériques** | `devmgmt.msc`    | Liste tous les périphériques installés, pilotes, conflits          |
| **Gestion de l’ordinateur**       | `compmgmt.msc`   | Accès à divers outils : disques, services, utilisateurs, etc.      |
| **Panneau de configuration**      | `control`        | Accès à l’ancien panneau de configuration Windows                  |
| **Explorateur de fichiers**       | `explorer`       | Ouvre une fenêtre de l’explorateur de fichiers                     |
| **Invite de commandes (Admin)**   | `cmd`            | Terminal classique Windows                                         |
| **PowerShell (Admin)**            | `powershell`     | Shell plus moderne avec plus de puissance pour l'automatisation    |
| **Exécuter**                      | `Win + R`        | Raccourci pour lancer rapidement toutes les commandes ci-dessus    |
| **Adaptateur réseau**             | `ncpa.cpl`       | Ouvre la fenêtre de configuration des adaptateurs réseaux.         |

---

🧠 **Astuce** : Pour accéder rapidement à ces outils, tape leur nom dans `Exécuter` (`Win + R`), dans le menu Démarrer, ou utilise un raccourci `.msc`.
